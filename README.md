# NWM - aNother Workspace Manager

## Installation

```npm i -g @nousantx/nwm```

### Usage

1. List all scripts

```sh
nwm list
```

2. Search scripts

```sh
nwm search <term>
```

Example :

```sh
nwm search build
```

3. Run a script

```sh
nwm run <workspace> <script>
```

Example :

```sh
nwm run core build
```

4. Interactive mode

```sh
nwm interactive
```
